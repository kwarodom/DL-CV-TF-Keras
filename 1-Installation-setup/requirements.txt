Download and Install anaconda:
https://www.continuum.io/downloads

Create a New Environment in Anaconda:
conda create -n tensorflow pip python=3.6 (Write the name as you want instead tensorflow)

Installing libraries by PIP in the new environment:
pip install pillow
pip install lxml
pip install jupyter
pip install matplotlib
pip install pandas
pip install -U scikit-learn
pip install h5py==2.8.0rc1
pip install beautifulsoup4
pip install requests
pip install Cython
python -m ipykernel install --user --name tf18 --display-name "TF1.8" (Write the name as you want instead ¨TF1.8¨)


Installing TensorFlow from:
https://www.tensorflow.org/install/install_windows

After Installing TensorFlow:
pip install keras
